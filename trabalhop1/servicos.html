Cadastro de Ingredientes

O cadastro de produtos permitirá que o usuário do sistema realize o cadastro de produtos variados no estoque.

Cadastro de Receitas

O cadastro de receita permitirá que o usuário realize o cadastro de diversas receitas culinárias no sistema.

Sugestão de Receitas

Esta funcionalidade permitirá que o usuário possa receber sugestões de receita com base nos produtos/receitas cadastradas com preferências da data de suas respectivas validades.

Cadastro de Listas de Compras

O sistema permitirá que o usuário defina a sua própria lista de compras que será utilizada como base para a funcionalidade abaixo.

Comparativo de Preços

O sistema fará um comparativo de valor de mercado a partir das listas criadas. Fazendo uma busca de preços dos mercados disponíveis no sistema retornando o comparativo entre os mesmos.
